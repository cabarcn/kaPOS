# KApos - Sistema de Gestión de Donaciones

Plataforma web multi-tenant para gestión de captación y suscripciones de socios.

## Stack Tecnológico

- **Backend:** Django 5.2.7 + Django REST Framework
- **Base de datos:** PostgreSQL
- **Autenticación:** JWT (Simple JWT)
- **Frontend:** React (próximamente)

## Requisitos Previos

- Python 3.10+
- PostgreSQL 13+
- Node.js 18+ (para el frontend)
